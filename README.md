code-coverage
======

> code-coverage - [php](http://php.net) command line tool for analysing [phpunit](https://phpunit.de) xml code coverage report

## Install
Via Composer

```sh
composer require g4/code-coverage
```

## Usage

## Development

### Install dependencies

    $ composer install

### Run tests

    $ make unit-tests
